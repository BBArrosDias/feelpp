Builds information around faces/elements for exporting data.

Parameters
meshThe mesh from which data is extracted
itStarting iterator over the faces/elements
enEndoing iterator over the faces/elements
outerIf false, the vertices are place in an x1 y1 z1 ... xn yn zn order, otherwise in the x1 ... xn y1 ... yn z1 ... zn
renumberIf true, the vertices will be renumbered with maps to keep the correspondance between the twoi, otherwise the original ids are kept
fillIt true, the method will generate points coordinates that are 3D, even if the point is specified with 1D or 2D coordinates (filled with 0)
Specifythe startIndex of the renumbered points (typically set to 0 or 1, but no restriction). This is only used when renumber is true, otherwise it is not used.
