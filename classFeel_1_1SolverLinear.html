Feel::SolverLinear< T, SizeT > Class Template Referenceabstract

Description

template<typename T, typename SizeT = uint32_type>
class Feel::SolverLinear< T, SizeT >

This class provides a uniform interface for linear solvers.

This base class is overloaded to provide linear solvers from different packages like FEEL, GMM or PETSC
