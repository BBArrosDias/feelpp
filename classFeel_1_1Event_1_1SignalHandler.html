Feel::Event::SignalHandler Class Reference

Description

Enable signals & slots to inherited objects. Add new functionnalities to connect objects between each others. Class that inherite from SignalHandler will have to possibility to create new signal.

:Note: Function template is a design choice. We want to simplify event creation in userland and also minimize polymorphism complexity.

:Info: This class is used to generate benchmark database.
